공공데이터 API 연동 문제
문제 인식
지질정보서비스시스템을 제외한 API는 사실상 민간인이 가져올 수 있는 데이터가 없어 데이터 전처리 과정에 어려움이 있었습니다.
문제 해결
지질정보서비스시스템에서 제공되는 지질상태의 정도를 로지스틱 회귀분석을 거쳐 위험도를 계산한 값, 한국수자원공사에서 제공하는 지하수 개발 가능량 지표로 지하수량의 비율을 Input Data로 적용하였습니다. 이후 데이터를 취합해 위험도가 1에 가까울 수록 높다고 기준을 정한 뒤, 상/중/하로 나눠 결과를 표시했습니다. 이를 통해 불가능이라 생각했던 문제도 색다른 접근법을 적용해 해결할 수 있는 역량을 갖추게 되었습니다.
